# הפינה המתוקה 🍦 — Sweet Corner

אתר React + Vite לחנות הגלידות "הפינה המתוקה" — RTL מלא בעברית.

##  מה יש באתר

- **עמוד בית קולנועי**: עיצוב מלא כרקע, ובמרכזו שקית המותג שנפתחת באנימציה תלת-ממדית (רעד → הבזק → הגלידות מתפרצות עם ניצוצות), הטיה 3D לפי העכבר וכפתור הפעלה מחדש
- **סל קניות מלא**: כפתור +1 על כל מוצר, אנימציית "טיסה לסל", מגירת סל עם פס משלוח חינם, קונפטי בסיום הזמנה, שמירה ב-localStorage
- **7 עמודים**: בית, טעמים (עם סינון), מבצעים, מארזים, אודות, צור קשר, סניפים

##  הרצה מקומית

```bash
npm install
npm run dev
```

האתר יעלה על http://localhost:5173

##  בנייה לפרודקשן

```bash
npm run build
```

##  מבנה

- `src/pages/` — עמודי האתר
- `src/components/` — רכיבים (BagHero = אנימציית השקית, CartDrawer = הסל, ועוד)
- `src/data/products.js` — כל המוצרים/מבצעים/סניפים — לעריכה קלה
- `public/images/` — תמונות (ראו README בתיקייה)
