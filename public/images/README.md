# תמונות מוצרים

## ⭐ תמונת עמוד הבית — הכי חשוב!

שמרו את תמונת העיצוב המלאה (זו מהצ'אט) כאן בשם **`home-hero.jpg`** —
וברגע שהיא קיימת, עמוד הבית יציג אותה במקום העיצוב המשוחזר,
כולל אזורי לחיצה שקופים מעל התפריט והכפתורים שבתוך התמונה.

איך לשמור: לחיצה ימנית על התמונה בצ'אט ← "שמור תמונה בשם" ←
בחרו את התיקייה הזו (`sweet-corner/public/images`) ואת השם `home-hero.jpg`.
(גם `png` שנשמר בשם `home-hero.jpg` יעבוד בדפדפן.)

שימו כאן את קבצי התמונות שלכם. האתר יציג אותן אוטומטית ברגע שהקובץ קיים —
עד אז מוצג placeholder מעוצב עם שם הקובץ הצפוי.
